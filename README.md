# Game 351 Assignment 1

## Project Overview

This project was created in Unity as part of Game 351 Assignment 1. The goal of the project was to create a realistic 3D environment featuring terrain sculpting, environmental assets, water systems, and interactive world design elements.

The environment includes:
- A large sculpted terrain with mountains, hills, plains, and depressions
- A castle area surrounded by a water moat
- A secondary pond/lake area with a shoreline and boat
- A spaceship crash site
- Vegetation, trees, and environmental decorations
- A controllable robot player character

---

# Implemented Features

## Terrain System
- Sculpted terrain using Unity Terrain tools
- Mountain ranges placed around the edge of the map to block visibility beyond the world
- Rolling hills and flatter interior regions
- Negative terrain relief including dips/craters

## Water Features
- Water moat surrounding the castle
- Secondary pond/lake area
- Shoreline terrain painting around water regions
- Realistic water shaders and textures

## Environment Design
- Castle environment
- Crash site with spaceship model
- Trees and vegetation throughout the world
- Environmental props and textures

## Player
- Robot player character
- Basic player movement and camera controls

## Visual Effects
- Water shaders/materials
- Terrain textures
- Environmental materials and visual effects

# Installation Procedure

1. Clone or download the project repository from GitHub.

2. Alternatively, click the **Download ZIP** option from the GitHub repository page.

3. Extract the ZIP folder if downloaded as a ZIP file.

4. Open Unity Hub.

5. Add the project folder to Unity Hub.

6. Open the project using Unity version:

```text
Unity 2021.3.5f1
```

7. Open the main scene located at:

```text
Assets/Scenes/Planet959 Scene
```

8. Press the Play button in the Unity Editor to run the project.

# Known Issues / Errata

- Water reflections may vary depending on graphics hardware.

- Performance may decrease on lower-end systems due to water and vegetation density.
- Some collision meshes are simplified for optimization purposes.
